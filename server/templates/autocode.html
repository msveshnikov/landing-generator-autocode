How It Works

Getting started with AutoCode is simple:

  1. Create a CLAUDE_KEY environment variable with your API key.
  2. Navigate to your project folder in the terminal.
  3. Run the following command:
npx autocode-ai

AutoCode will then read your README.md file and existing source code, interpret the instructions using the Claude 3.5 Sonnet API, and generate the necessary code structure, files, and content. It's that easy!
